WebRTC, Mengirimkan SDP (Session Description Protocol) tanpa signaling server.

Instruksi: Mulai dengan membuka dua tab pada browser secara bersebelahan dan ikuti langkah-langkah di bawah ini untuk mengirimkan SDP offer dan SDP answer. Kita andaikan bahwa setiap tab sebagai User 1 dan User 2.

Source Code

Step 1: User 1, klik tombol "Create offer" untuk membuat SDP offer dan copy offer dari text area di bawah ini.

Step 2: User 2, paste SDP offer yang dibuat oleh user 1 ke dalam text area di bawah, kemudian klik "Create Answer" membuat SDP answer dan copy SDP answer tersebut dari text area di bawah.

Step 3: User 1, paste SDP offer yang dibuat oleh user 2 ke dalam text area di bawah kemudian klik "Add Answer"
